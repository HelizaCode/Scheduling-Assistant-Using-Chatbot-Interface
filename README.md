## 🧠 AI-Powered Conversational Schedule Planner

A full-stack web application that reimagines calendar management through a **chat-driven interface**, allowing users to plan, organize, and optimize their schedules using natural language.

Instead of relying on traditional form-based inputs, this system enables users to interact with their schedule conversationally—bridging the gap between **human intent and structured time management**.

---

## ✨ Highlights

* 💬 **Conversational Scheduling Interface**
  Manage your calendar by simply typing commands like:

  * *“Add meeting with Alex tomorrow at 3pm”*
  * *“What’s on my schedule today?”*
  * *“Move my 2pm event to Friday at 10am”*

* 🧠 **Natural Language Understanding**
  Interprets flexible human input including:

  * Relative dates (*today, tomorrow, next week*)
  * Time formats (*3pm, 14:30*)
  * Duration (*1 hour, 90 minutes*)

* 📅 **Dynamic Calendar Visualization**

  * Day and week views with a structured time grid
  * Real-time rendering of events
  * Clear visual distinction for scheduled vs suggested time slots

* ⚡ **Smart Scheduling Engine**

  * Detects conflicts and overlaps
  * Suggests optimal free time slots for new activities
  * Dynamically updates schedule based on user commands

---

## 🏗️ System Design

The application integrates three core components:

* **Chat Interface**
  Handles user interaction and captures intent through conversational input.

* **Parsing & Logic Layer**
  Processes natural language using rule-based parsing to extract:

  * Event title
  * Date & time
  * Duration

* **Scheduling Engine**
  Maintains event data, resolves conflicts, and identifies available time gaps.

* **Calendar UI**
  Visually represents scheduled events and suggested slots in a responsive grid layout.
